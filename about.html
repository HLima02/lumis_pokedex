Pokedex API

A Pokédex é uma aplicação interativa criada para explorar o universo Pokémon de forma simples, rápida e visualmente agradável. Aqui, você pode descobrir informações detalhadas sobre cada Pokémon, explorar seus tipos, habilidades e características, além de utilizar ferramentas de busca e filtros para encontrar exatamente o que procura.

Este projeto foi desenvolvido com foco em performance, organização de dados e experiência do usuário, consumindo informações diretamente da PokéAPI, uma API pública amplamente utilizada para estudos e projetos educacionais.

Seja você um treinador Pokémon curioso ou um desenvolvedor explorando novas habilidades, esta Pokédex foi pensada para oferecer uma experiência clara, intuitiva e envolvente.
